
# 🧑‍💻 Benutzerinterface (Eltern & Schüler)

Willkommen in der Anleitung für das MensaPay-Benutzerportal. Dieses Interface wurde speziell für Eltern und Schüler entwickelt, um die Schulverpflegung einfach, transparent und bargeldlos zu verwalten.

## 1. Registrierung & Login

-   **Account erstellen:** Eltern können sich mit ihrem Vor- und Nachnamen sowie einer E-Mail-Adresse registrieren. Das Passwort muss strenge Sicherheitskriterien erfüllen (mind. 8 Zeichen, Groß-/Kleinbuchstaben, Zahl, Sonderzeichen).
    
-   **Login:** Der Login erfolgt über E-Mail und Passwort. Zum Schutz vor Spam und automatisierten Angriffen ist ein Captcha ("Ich bin ein Mensch") vorgeschaltet.
    
-   **Passwort vergessen:** Über die "Passwort vergessen"-Funktion kann jederzeit ein sicherer Reset-Link per E-Mail angefordert werden.
    

## 2. Dashboard & Guthaben-Management

Das Dashboard ist die zentrale Startseite nach dem Login. Hier siehst du sofort das **aktuelle Familienguthaben**.

**Guthaben aufladen:** Über den Button `Guthaben aufladen` stehen drei Wege zur Verfügung:

1.  **Online (PayPal / Klarna):** Der Betrag (z.B. 10€, 20€, 50€) wird direkt gutgeschrieben und steht sofort zur Verfügung.
    
2.  **Banküberweisung (Vorkasse):** Das System generiert eine eindeutige PIN (z.B. `MENSA A1B2C`). Überweise den Betrag mit dieser PIN im Verwendungszweck an die Schule. Das Guthaben wird gutgeschrieben, sobald die Schule den Geldeingang verbucht hat (ca. 1-3 Werktage).
    
3.  **Barzahlung vor Ort:** In der Finanzstelle der Schule kann gegen Nennung der Account-E-Mail jederzeit bar eingezahlt werden (sofortige Gutschrift).
    

## 3. Abo-Shop (Mensabos)

Im Reiter "Abos" können regelmäßige Essensabos abgeschlossen werden.

-   **Abo-Typ wählen:** Wähle zwischen einem **Halbjahresabo** oder einem **Ganzjahresabo**.
    
-   **Wochentage festlegen:** Wähle exakt die Tage (z.B. Dienstag und Donnerstag), an denen das Kind in der Mensa essen wird. Der Preis wird dynamisch pro Wochentag berechnet.
    
-   **Schülerprofil:** Weise das Abo einem bestehenden Schülerprofil zu oder lege im gleichen Schritt ein komplett neues Schülerprofil (inkl. Kartenpfand) an.
    
-   **Bezahlung:** Abos können mit bestehendem Guthaben verrechnet werden. Offene Restbeträge lassen sich via PayPal, Klarna oder Überweisung begleichen.
    

## 4. Kartenverwaltung (Chipkarten)

Im Reiter "Karten" siehst du alle angelegten Schülerprofile und den Status der dazugehörigen Chipkarten.

-   **Neue Karte bestellen:** Du kannst jederzeit eine reine Prepaid-Karte für ein weiteres Kind bestellen (Kartenpfand wird fällig).
    
-   **Karte verloren / Temporär sperren:** Wurde eine Karte verlegt, kann sie mit einem Klick auf `Sperren` sofort deaktiviert werden. Sie ist dann am Mensa-Terminal unbrauchbar.
    
-   **Ersatzkarte beantragen:** Ist die Karte endgültig weg, klicke auf `Karte verloren` -> `Sperren & Neu beantragen`. Die alte Karte wird gelöscht und gegen eine Gebühr (Pfand) eine neue Ersatzkarte im System hinterlegt. _Hinweis: Die physische Ersatzkarte muss vom Schüler im Sekretariat/bei der Lehrkraft abgeholt werden._
    

## 5. Transaktionshistorie

Auf dem Dashboard findest du eine detaillierte Liste aller Geldbewegungen:

-   Aufladungen (Grün markiert, mit Datum und Betrag)
    
-   Ausgaben für Abos oder Kartenpfand
    
-   (Zukünftig: Ansicht der einzelnen Essensabbuchungen direkt in der Mensa)
